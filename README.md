### 💾 Installation

```bash
# клонировние репозитория
git clone https://github.com/pi10el/messager-server

# установка зависимостей проекта
$ npm install

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
